##   属性  
 
| 参数    | 说明      | 类型 |
| ------- |---------|-----| 
|  rowIndex  |  行索引  |  number  |  
|  row  |  行数据  |  object  | 
|  base_table  |  查询列表的表格配置  |  object  | 
 
**示例**
```javascript

console.info(this.rowIndex); // 控制台输出行索引
console.info(this.row); // 控制台输出行数据
console.info(this.base_table); // 控制台输出查询列表的表格配置

```
