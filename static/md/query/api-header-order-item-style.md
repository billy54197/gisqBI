##   属性

| 参数    | 说明      | 类型 |
| ------- |---------|-----|  
|  config  |  当前配置  |  object  |   
|  itemStyle  |  已经配置过的排序结点样式  |  object  |
|  iconStyle  |  已经配置过的图标样式  |  object  |


**示例**
```javascript

console.info(config);
console.info(itemStyle);
console.info(iconStyle);

itemStyle.backgroundColor = "#f5f7fa"; // 设置背景颜色
iconStyle.backgroundColor = "#f5f7fa"; // 设置背景颜色


```
