##   属性

| 参数    | 说明      | 类型     |
| ------- |---------|--------| 
|  requestData  |  绑定的数据请求的数据  | object |  
|  label  |  根节点  | 绑定的数据请求对应的字段内容 |   
|  description_config  |  统计说明的配置信息  | object |   


##   方法

| 方法名    | 说明      | 参数 |
| ------- |---------|-----| 
|    |    |    |   




**示例**
```javascript

console.info(label) // 打印字段内容;
console.info(requestData); // 数据请求的数据

return label;

```
