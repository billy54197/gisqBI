|    |    |    |  
 

 ##   方法  
  
| 方法名    | 说明      | 参数 |
| ------- |---------|-----| 
|  doQuery()  |  查询结果  |    |  
|  emptyHeader()  |  清空头部查询条件  |    |  
|  doRefresh()  |  刷新结果  |    |  
|  doExportExcelCurrent  |  导出Excel（当前页）  |    |  
|  doExportExcelAll()  |  导出Excel（所有）  |    |  



**示例**
```javascript

this.doQuery(); // 查询结果
this.emptyHeader(); // 清空头部查询条件
this.doRefresh(); // 刷新结果
this.doExportExcelCurrent(); // 导出Excel（当前页）
this.doExportExcelAll(); // 导出Excel（所有）

```
