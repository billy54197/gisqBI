##   属性  
 
| 参数    | 说明      | 类型 |
| ------- |---------|-----|  
|  config  |  当前配置  |  object  |   
|  style  |  已经配置过的样式  |  object  |
 

**示例**
```javascript
 
console.info(config);
console.info(style);

style.backgroundColor = "#f5f7fa"; // 设置背景颜色
style.lineHeight = "40px"; // 设置行高
style.paddingTop = "5px"; // 设置上边距
style.backgroundImage = "url('/gisqBI/image.png')"

```
